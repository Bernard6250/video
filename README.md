# Galerie Vidéo Formation

Ce projet est une interface web moderne permettant de visionner des vidéos de formation hébergées sur Google Drive et d'accéder à des ressources Notion.

## Comment utiliser

1. Ouvrez simplement le fichier `index.html` dans votre navigateur web (Chrome, Firefox, Edge, etc.).
2. Une connexion internet est requise pour charger les vidéos et accéder aux ressources.

## Contenu

- **Session 1**: Vidéo d'introduction.
- **Session 2**: Vidéo sur Sora Pro.
- **Ressources**: Lien vers le guide complet sur Notion.

## Avertissement

Les vidéos sont hébergées sur Google Drive. Si elles ne s'affichent pas ("Accès refusé" ou similaire), assurez-vous que les fichiers sur le Drive source sont bien partagés en mode **"Tous les utilisateurs disposant du lien"** (Lecteur).
